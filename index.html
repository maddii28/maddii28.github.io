Finlitify Github Repository

Finlitify utilizes Computer Adaptive Testing (CAT) and integrates the Wolfram Alpha Short Answers API to dynamically adjust quiz difficulty based on user performance. The platform personalizes financial literacy content across budgeting, savings, and taxes by adapting lessons and assessments to individual knowledge levels, and was awarded the Best Use of Wolfram Alpha in HackNYU 2023.

Python Short Answers API React

Lockabox

Lockabox, developed as a class project for EG-UY 1004: Introduction to Engineering and Design, is a security device that integrates a Raspberry Pi, TensorFlow-powered text detection, solenoid locking mechanism, beeper, and LED-touch screen. It prevents distractions while ensuring safety through emergency access capabilities.

Python TensorFlow Raspberry Pi Solenoid Lock

QRavenGithub Repository

QRaven is a dynamic QR code platform that uses Flask for backend development and integrates machine learning for context-based content delivery. It adapts to user-defined triggers like location and time, providing personalized, real-time responses via APIs.
